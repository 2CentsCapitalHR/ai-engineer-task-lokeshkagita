## Setup

1. Install dependencies:

2. Replace YOUR_GEMINI_API_KEY in `doc_analyzer.py`.

3. Run the app:
   python app.py

4. Upload `.docx` legal files.
5. Outputs will be saved in the `outputs/` folder.
